# Sebastian Claesson - Techblog

A techblog about Azure, PowerShell, automation and IT security, written for IT pros and architects.

**Live at:** [blog.sebastianclaesson.se](https://blog.sebastianclaesson.se)

## Structure

```
.
├── index.html          # Home page with post listing, search & category filter
├── about.html          # About page
├── posts/              # Blog posts (HTML)
├── css/style.css       # Styling (dark/light mode)
├── js/main.js          # Search, filtering, theme toggle
├── assets/images/      # Post images and avatar
├── sitemap.xml         # SEO sitemap
├── robots.txt          # SEO robots
└── .nojekyll           # Tells GitHub Pages to skip Jekyll processing
```

## Stack

Pure static HTML/CSS/JS. No build step, no framework dependencies. Deployed to GitHub Pages via GitHub Actions.

## Local preview

Open `index.html` in a browser.
