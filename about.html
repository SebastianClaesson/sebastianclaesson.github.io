About

TechLead, Platform Engineering - Azure, PowerShell & Security

Welcome to my techblog!

My name is Sebastian. I live and work in Uppsala, Sweden, where I'm currently a TechLead for the Platform Engineering team at Viedoc with primary focus on PowerShell, Security and Infrastructure-as-Code (Bicep) within Azure.

This blog contains my personal thoughts and write-downs regarding issues and solutions, which hopefully will help other geeks like myself.

Uppsala Azure User Group

I run the Uppsala Azure User Group (UAUG) together with Björn Sundling. We host free meetups in Uppsala where we bring tech people together to share knowledge and stay up to date with the latest in the Microsoft cloud sphere. Open to everyone, from beginners to seasoned cloud architects.

Open Source

I actively contribute to a few open source projects:

  • Maester - Security test automation framework for Microsoft 365 and Azure
  • AZDOPS - PowerShell module for managing Azure DevOps
  • NetboxTools - PowerShell module for Netbox network infrastructure management
  • Azure Community Policy - Community-contributed Azure Policy definitions
